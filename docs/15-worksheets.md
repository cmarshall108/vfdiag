## 15.2 HV Service De-Energize / Re-Energize Checklist

**VIN:** ______________________________ **Date:** _____________
**Lead tech:** ______________ **Verifier:** ______________

### De-Energize
- [ ] PPE donned: Class 0 (1000 V) gloves + leather over, face shield, arc-rated clothing
- [ ] Vehicle parked, P, EPB on, wheels chocked
- [ ] Key fob removed ≥ 5 m away
- [ ] Waited ≥ 2 min for buses to sleep
- [ ] 12 V negative disconnected, terminal isolated
- [ ] MSD pulled and tagged with technician's lock
- [ ] Waited ≥ 5 min for capacitor bleed
- [ ] DMM verified (CAT III 1000 V):
  - HV+ to HV− = ____ V (target ≤ 5 V)
  - HV+ to chassis = ____ V (target ≤ 5 V)
  - HV− to chassis = ____ V (target ≤ 5 V)
- [ ] LOTO tag applied

### Re-Energize
- [ ] All HV connectors fully seated (audible click), HVIL routing clear
- [ ] HV cabling not pinched, properly clipped, no orange jacket damage
- [ ] MSD reinstalled, latch fully secured
- [ ] 12 V negative reconnected, terminal torque ____ N·m
- [ ] Vehicle waked — no fault sounds, no smoke
- [ ] Full system scan — record/clear logistical U-codes
- [ ] Isolation reading via scan tool ≥ ____ Ω/V (target ≥ 100)
- [ ] Test drive ≥ 10 min — no DTCs after
